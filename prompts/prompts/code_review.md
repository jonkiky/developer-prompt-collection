## Tips for Better Code Review Prompts

1. Provide context:
   - Project background
   - Team standards
   - Performance requirements
   - Security needs

2. Specify review focus:
   - Critical areas
   - Known concerns
   - Time constraints
   - Required depth

3. Ask for:
   - Concrete examples
   - Alternative solutions
   - Trade-off analysis
   - Learning resources
